# Metrics and Verification

ORCS uses a layered measurement model.

## 1. Established infrastructure metrics
- PUE
- WUE
- CUE

These are established and useful, but incomplete.

## 2. Public research efficiency proxies
- joules per token
- tokens per watt
- energy per correct solution
- related benchmark-specific capability-efficiency measures

These are useful public proxies, but not yet one universally accepted intelligence-efficiency standard.

## 3. Proposed ORCS metrics
- Watershed Days
- local circularity reporting
- regenerative pilot disclosure schemas

These are proposed extensions intended to improve long-horizon substrate accounting.

## 4. Evidence discipline
Every metric in ORCS documentation should be identified as:
- verified
- derived
- or proposed
