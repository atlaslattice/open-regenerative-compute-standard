## Element 145 — Meta-Orchestrator Routing Logic

Element 145 does not compete for sphere primacy. It sits outside the 144-sphere grid and performs:

1. **Query Classification** — Map incoming query to sphere(s) using the ontology
2. **Primary Lookup** — Route to highest-weighted model for that sphere
3. **Tier Assignment** — Fast Path (1 model) / Standard Path (2-3 models) / Deep Path (full council)
4. **Adversarial Review** — For Standard and Deep paths, route to secondary for cross-check
5. **Provenance Tracking** — Log which model answered, which reviewed, confidence levels
6. **Conflict Resolution** — When primary and secondary disagree, escalate to convenor or third model
