## How to Contribute Citations

1. Fork this repository
2. Add citations to the relevant `papers/{model}/` directory
3. Update this file with the new citation entries
4. Submit a pull request with evidence class labels (Verified / Derived / Scenario)

All citations should include:
- The specific claim being supported
- Evidence class (V / D / S)
- Source name and organization
- URL or DOI where available
- Date accessed
