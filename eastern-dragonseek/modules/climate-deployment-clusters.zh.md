# 气候部署集群：中国区域优化方案

*2026年4月 | CC BY-SA 4.0*

---

## 概述

本文件将ORCS的六大气候部署集群映射到中国具体区域，提供本地化的模块组合建议。

---

## 集群A：干旱大陆型 / Arid Continental

**适用区域：** 内蒙古（呼和浩特、乌兰察布）、甘肃（庆阳）、宁夏、新疆

**气候特征：** 年降水量200-400mm，冬季极寒（-30°C），夏季炎热干燥，蒸发量远大于降水量

**核心挑战：** 水资源极度稀缺，风沙侵蚀，冬季供暖需求大

**推荐模块组合：**

| 优先级 | 模块 | 理由 |
|--------|------|------|
| 必选 | M2.6 地下植被床 | 零蒸发热消纳+防沙治沙 |
| 必选 | M1.1 蒸汽回收 | 最大化水回收 |
| 必选 | M5.2 AI水文学家 | 优化稀缺水资源分配 |
| 推荐 | M6.1 寒冷气候选址 | 利用自然冷源 |
| 推荐 | M2.4 区域供暖 | 冬季废热供暖 |
| 推荐 | M4.5 坎儿井预冷 | 新疆特有，零能耗预冷 |
| 可选 | M1.4 大气水发生器 | 补充水源 |
| 可选 | M3.7 含水层回灌 | 恢复地下水位 |

**政策对接：** 东数西算工程、防沙治沙法、黄河流域生态保护和高质量发展

**预期流域生命日：** +120至+200天/年

## 集群B：亚热带湿润型 / Subtropical Humid

**适用区域：** 贵州（贵安新区）、重庆、湖南、广西

**气候特征：** 年降水量1000-1500mm，湿度高（70-90%），温和冬季，喀斯特地貌

**核心挑战：** 高湿度增加冷却能耗，喀斯特地下水系统脆弱，季节性暴雨

**推荐模块组合：**

| 优先级 | 模块 | 理由 |
|--------|------|------|
| 必选 | M4.2 海绵园区 | 管理充沛降水 |
| 必选 | M1.5 雨洪捕获 | 利用丰富降水 |
| 必选 | M3.3 人工湿地 | 适合湿润气候的水处理 |
| 推荐 | M2.7 菌菇培养 | 利用湿度和废热 |
| 推荐 | M1.7 冷凝水回收 | 高湿度环境产出大 |
| 推荐 | M4.7 生物多样性缓冲区 | 保护喀斯特生态 |
| 可选 | M2.1 温室农业 | 补充当地食品供应 |
| 可选 | M5.4 水质监测 | 喀斯特水质敏感 |

**政策对接：** 大数据谷建设、海绵城市试点、喀斯特石漠化治理

**预期流域生命日：** +200至+365天/年（降水充沛，净正效益潜力大）

## 集群C：温带季风型 / Temperate Monsoon

**适用区域：** 张北（河北）、北京周边、山东、辽宁

**气候特征：** 年降水量500-800mm，四季分明，冬季寒冷，夏季炎热湿润

**核心挑战：** 冬季供暖需求大，夏季冷却需求高，水资源季节性分布不均

**推荐模块组合：**

| 优先级 | 模块 | 理由 |
|--------|------|------|
| 必选 | M2.4 区域供暖 | 冬季废热利用 |
| 必选 | M6.3 市政热互惠 | 换取政策支持 |
| 必选 | M1.1 蒸汽回收 | 夏季冷却塔水回收 |
| 推荐 | M5.1 公开仪表板 | 京津冀环保要求高 |
| 推荐 | M4.2 海绵园区 | 管理夏季暴雨 |
| 推荐 | M2.1 温室农业 | 冬季蔬菜供应 |
| 可选 | M3.5 流分离回收 | 高标准水回用 |
| 可选 | M4.3 水权交易 | 华北水权市场活跃 |

**政策对接：** 京津冀协同发展、清洁取暖、冬奥绿色遗产

**预期流域生命日：** +100至+200天/年

## 集群D：热带海洋型 / Tropical Maritime

**适用区域：** 海南、深圳、广州、厦门

**气候特征：** 年降水量1500-2500mm，全年高温高湿，台风季节

**核心挑战：** 极高冷却能耗，台风风险，海水腐蚀

**推荐模块组合：**

| 优先级 | 模块 | 理由 |
|--------|------|------|
| 必选 | M1.6 海水热交换 | 利用海水冷却 |
| 必选 | M3.6 先进膜脱盐 | 海水淡化补充 |
| 必选 | M1.5 雨洪捕获 | 利用充沛降水 |
| 推荐 | M2.3 藻类培养 | 热带气候适合 |
| 推荐 | M4.7 生物多样性 | 热带生态敏感 |
| 推荐 | M1.4 大气水发生器 | 高湿度产出大 |
| 可选 | M2.2 水产养殖 | 热带鱼虾养殖 |
| 可选 | M4.8 零液体排放 | 海洋环保要求 |

**政策对接：** 海南自由贸易港、粤港澳大湾区、海洋生态保护

**预期流域生命日：** +250至+400天/年

## 集群E：高原型 / Plateau

**适用区域：** 青海、西藏、云南（高海拔区域）

**气候特征：** 高海拔（3000m+），低温，强紫外线，降水少但冰川融水

**核心挑战：** 空气稀薄影响风冷效率，生态极度脆弱，基础设施有限

**推荐模块组合：**

| 优先级 | 模块 | 理由 |
|--------|------|------|
| 必选 | M6.1 高海拔选址 | 自然冷源充足 |
| 必选 | M4.7 生物多样性 | 高原生态极脆弱 |
| 必选 | M5.3 含水层监测 | 冰川融水变化 |
| 推荐 | M2.6 地下植被床 | 高原植被恢复 |
| 推荐 | M1.5 雨洪捕获 | 收集冰川融水 |
| 可选 | M2.1 温室农业 | 高原蔬菜供应 |

**政策对接：** 三江源保护、青藏高原生态屏障、清洁能源基地

**预期流域生命日：** +80至+150天/年（生态脆弱性乘数高）

## 集群F：长江中下游型 / Yangtze Middle-Lower

**适用区域：** 上海、南京、武汉、杭州、合肥

**气候特征：** 年降水量1000-1400mm，梅雨季节，夏季酷热（"火炉"城市）

**核心挑战：** 夏季极端高温，梅雨季洪涝，城市热岛效应

**推荐模块组合：**

| 优先级 | 模块 | 理由 |
|--------|------|------|
| 必选 | M4.2 海绵园区 | 梅雨季管理 |
| 必选 | M2.4 区域供暖/制冷 | 冬季供暖+夏季余热驱动吸收式制冷 |
| 必选 | M3.4 MBR水处理 | 城市废水回用 |
| 推荐 | M6.3 市政互惠 | 城市政策对接 |
| 推荐 | M1.5 雨洪捕获 | 梅雨季水资源化 |
| 推荐 | M5.1 公开仪表板 | 长三角环保要求高 |
| 可选 | M1.6 湖泊热交换 | 利用太湖/巢湖等 |
| 可选 | M4.3 水权交易 | 长江经济带水权 |

**政策对接：** 长三角一体化、长江经济带生态保护、海绵城市

**预期流域生命日：** +150至+300天/年

---

## 总结

中国的地理和气候多样性意味着不存在"一刀切"的再生计算方案。本集群指南提供了区域化的起点，但每个具体站点都需要基于当地水文、政策和社区条件进行定制化设计。

**关键原则：** 先测量，再设计，持续监测，公开数据。
