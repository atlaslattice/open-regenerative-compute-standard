# 流域生命日与生态产品价值实现（EPVR）桥接方法论

## 1. 目的

本文件提供将ORCS的**流域生命日**指标桥接到中国**生态产品价值实现（EPVR）**框架的完整计算方法论。该桥接使数据中心运营商能够将水文绩效转化为绿色金融资格。

## 2. EPVR框架概述

生态产品价值实现（EPVR）是生态环境部的旗舰框架，用于量化生态系统服务的经济价值。自2021年起，多个省份已开展GEP（生态系统生产总值）核算试点。

EPVR涵盖四大类生态产品：

| 类别 | 包含内容 | 与ORCS的关联 |
|------|---------|-------------|
| 物质供给 | 水资源、食品、木材 | 净水产出、设施农业产出 |
| 调节服务 | 水源涵养、气候调节、水土保持 | 含水层补给、碳封存、防沙化 |
| 文化服务 | 景观、教育、科研 | 开放数据、社区教育 |
| 支持服务 | 生物多样性、土壤形成 | 生态修复、微生物激活 |

## 3. 桥接计算步骤

### 步骤1：计算原始流域生命日

按照 `deepseek/watershed-days-metric.zh.md` 中的方法论计算原始流域生命日（WD_raw）。

### 步骤2：应用公平权重乘数

```
WD_weighted = WD_raw × 干旱指数乘数 × 社区脆弱性乘数
```

| 干旱等级 | 乘数 | 中国气象局等效 |
|---------|------|--------------|
| 无旱 | 1.0x | 正常 |
| 轻旱 | 1.3x | 气象干旱黄色预警 |
| 中旱 | 1.7x | 气象干旱橙色预警 |
| 重旱 | 2.3x | 气象干旱红色预警 |
| 特旱 | 3.0x | 特大干旱 |

### 步骤3：转换为EPVR货币价值

使用省级GEP系数将加权流域生命日转换为人民币价值：

```
EPVR_value (元) = WD_weighted × 省级水资源GEP系数 × 设施服务面积(km²)
```

参考省级GEP系数（2024年数据）：

| 省份 | 水资源GEP系数（元/天·km²） | 备注 |
|------|--------------------------|------|
| 内蒙古 | 12,000-18,000 | 干旱区，水资源稀缺性高 |
| 贵州 | 8,000-12,000 | 喀斯特地貌，水文复杂 |
| 北京 | 25,000-35,000 | 城市区域，水资源极度稀缺 |
| 江苏 | 6,000-9,000 | 温带，水资源相对充裕 |
| 新疆 | 15,000-22,000 | 极端干旱区 |
| 海南 | 5,000-8,000 | 热带，降水充沛 |

*注：以上系数为保守估计，实际值需根据省级GEP核算结果确定。*

### 步骤4：映射到绿色金融资格层级

| 加权流域生命日 | EPVR层级 | 金融工具 | 预估利率优惠 |
|--------------|---------|---------|------------|
| +100至+200 | 基础生态贡献者 | 可持续发展挂钩贷款折扣 | 0.1-0.3%利率降低 |
| +200至+365 | 显著生态贡献者 | 绿色债券资格 | 0.3-0.5%利率降低 |
| +365以上 | 净正效益生态锚点 | 优先生态补偿+绿色债券 | 0.5-1.0%利率降低+直接补贴 |

### 步骤5：审计与验证

EPVR价值声明的验证要求：

1. **数据来源：** 所有水文计量数据必须来自经校准的流量计，校准记录公开可查。
2. **第三方核查：** 由具有CMA资质的第三方机构进行年度核查。
3. **省级备案：** EPVR计算结果需向省级生态环境厅备案。
4. **公开披露：** 计算方法论和原始数据通过ORCS仓库公开。

## 4. 示例计算：内蒙古10MW节点

| 步骤 | 参数 | 值 |
|------|------|-----|
| 原始流域生命日 | WD_raw | +86天/年 |
| 干旱指数 | D2（中旱） | 1.7x |
| 社区脆弱性 | 单一主要水源 | 1.3x |
| 加权流域生命日 | WD_weighted | +190天/年 |
| 省级GEP系数 | 内蒙古 | 15,000元/天·km² |
| 服务面积 | 15km半径 | ~707 km² |
| **年度EPVR价值** | | **约20.2亿元** |
| 绿色金融层级 | 基础生态贡献者 | 可持续发展挂钩贷款资格 |

*注：以上为说明性计算。实际EPVR价值需经省级核算确认。*

## 5. 与现有绿色金融标准的对接

| 标准 | 对接方式 |
|------|---------|
| 中国绿色债券标准委员会目录（2021版） | 流域生命日+200以上的设施可归入"生态环境产业"类别 |
| 人民银行绿色金融评价方案 | EPVR价值可作为绿色贷款评价的定量依据 |
| CCER（中国核证自愿减排量） | 防沙化种植（SSVB模块）产生的碳汇可申请CCER |
| 生态补偿转移支付 | 净正效益设施可申请省级生态补偿资金 |

## 6. 认知状态声明

本方法论中的GEP系数为保守估计，基于公开的省级试点数据。实际部署前，必须与省级生态环境厅确认当地GEP核算标准和系数。绿色金融资格层级为指导性建议，最终资格由金融机构根据具体项目评估确定。
